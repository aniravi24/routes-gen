# @routes-gen/remix

This is package is a **routes-gen** driver for [Remix](https://github.com/remix-run/remix).

## Installation

```
yarn add routes-gen
yarn add -d @routes-gen/remix
```

## Usage

```
yarn routes-gen -d @routes-gen/remix
```

By default, it exports the routes to: `/app/routes.d.ts`.
